# Collaborative Canvas

A real-time collaborative drawing app using vanilla JS and Node.js WebSocket.

## Setup

1. Install dependencies:
   `npm install`
2. Start the server:
   `npm start`
3. Open `client/index.html` in your browser (use Live Server or similar)

## Usage

- Open the app in multiple browser tabs/windows to simulate collaboration.
- Draw and see updates in real time!
